# EcomX - Scalable BDD-Based UI & API Automation Framework

## Overview

EcomX is an enterprise-style automation framework designed to automate critical e-commerce workflows through both UI and API layers.

## Technology Stack

* Java 17
* Selenium 4
* Cucumber BDD
* TestNG
* Rest Assured
* Maven
* Jenkins
* Docker

## Features

* UI Automation
* API Automation
* BDD Implementation
* Page Object Model
* Driver Factory
* Reporting
* Parallel Execution
* CI/CD Integration
