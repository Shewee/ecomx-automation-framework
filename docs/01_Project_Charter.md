# EcomX - Scalable BDD-Based UI & API Automation Framework

## Objective

The objective of EcomX is to design and implement a scalable, maintainable, and enterprise-ready automation framework capable of supporting both UI and API automation.

## Technology Stack

* Java 17
* Selenium 4
* Cucumber BDD
* TestNG
* Rest Assured
* Maven
* Extent Reports
* Jenkins
* Docker

## Goals

* UI Automation
* API Automation
* BDD Implementation
* CI/CD Integration
* Parallel Execution
* Reporting

## Expected Outcome

A reusable enterprise-grade automation framework demonstrating Automation Lead and Automation Architect principles.
